스크린샷 캡쳐

전체화면 캡쳐

command
+
shift
+
3
  1. command키와 shift키를 동시에 누른 상태로 숫자 3번 키를 누른다.
  2. 모니터 우측 하단에 생성된 캡쳐 축소판을 클릭하여 편집하거나 자동저장(4~5초)이 될 때까지 기다린다.

일부 화면 캡쳐

command
+
shift
+
4
  1. command키와 shift키를 동시에 누른 상태로 숫자 4번키를 누른다.
  2. 십자선을 마우스로 클릭 후 드래그 하여 캡쳐할 영역을 지정 후 축소판을 클릭하여 편집하거나 자동 저장(4~5초)이 될 때까지 기다린다.

윈도우 또는 메뉴 캡쳐

command
+
shift
+
4
+
space
  1. 캡쳐할 윈도우 또는 메뉴를 화면상에 보이게 해놓는다.
  2. command키와 shift키를 동시에 누른 상태로 숫자 4번 키를 누르고 space키를 누른다.
  3. 마우스 포인터가 카메라 아이콘으로 변경된다.
  4. 스크린샷을 찍을 윈도우 또는 메뉴를 선택한 후 축소판을 클릭하여 편집하거나 자동 저장(4~5초)이 될 때까지 기다린다.

전체화면 시작 및 종료

fn
+
F
control
+
command
+
F
  1. fn키를 누른 상태로 f 키를 누른다.
  2. 또는 control키와 command키를 누른 상태로 f 키를 누른다.
  3. 전체화면 상태라면 종료가 되고 전체화면이 아니라면 전체화면으로 전환된다.

바탕화면 보기

command
+
F3
  1. command키를 누른 상태로 F3 키를 누른다.
  2. 다시 작업 화면으로 돌아가려면 동일하게 누르면 된다.
