URLSearchParams 객체 사용

URL의 파라미터 값을 추출해야 할 경우가 생깁니다.

파마리터의 값을 가지고 분기처리를 할 수도 있고 해당 페이지 내부의 링크값을 변경할 수도 있는 등등 많은 곳에서 사용이 되고 있습니다.

모든 파라미터 구하기


// URL 예시 : https://tistory.com/?name=apple&color=red

// 현재 페이지의 URL을 가져옵니다.
var urlParams = new URLSearchParams(window.location.search);

// 모든 파라미터의 키-값 쌍을 출력합니다.
for (var [key, value] of urlParams) {
    console.log(key, value);
    // name apple
    // color red
}

특정 파마리터 구하기


// URL 예시 : https://tistory.com/?name=apple&color=red

// 현재 페이지의 URL을 가져옵니다.
var urlParams = new URLSearchParams(window.location.search);

console.log(urlParams.get('name')); // apple
console.log(urlParams.get('color')); // red
